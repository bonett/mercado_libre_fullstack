## How to run this App

1. Go to the server folder of the repo.

2. Create a `.env` file with the global settings for the app.

3. Install Packages (Node Modules).

   ```
   yarn install
   ```

4. Run the app.

   ```
   yarn start
   ```

5. Finally, you see the project running on Port 'localhost:3000'
